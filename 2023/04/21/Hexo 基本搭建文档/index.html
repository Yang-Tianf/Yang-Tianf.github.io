Hexo 基本搭建文档

Hexo 是一款快速、简洁且高效的博客框架，可以让用户方便地写作和发布博客。以下是基本的 Hexo 搭建操作步骤：

1. 在终端或命令行中执行以下命令安装 Hexo

$npm install hexo-cli -g

2. 创建一个空的目录

$cd myblog cd  /* 进入你创建的目录 */

3. 在目录中初始化 Hexo

$hexo init

4. 安装所需的 Hexo 插件

$npm install

5. 启动服务器并预览网站

$hexo server
print("hello world")

6. 打开浏览器，输入 localhost:4000 即可访问您的博客

以上是快速基本的 Hexo 搭建操作，更多详细内容可以到 Hexo 官网(https://hexo.io/)
